格式建议
  • 本站使用 Markdown 格式，想了解这一格式请阅读：《用Markdown语法写文章》
  • 内容框下有实时预览框，请预览后发帖。
  • 文字前面请不要空4个英文空格
  • 每段文字之间请回两次车
  • 贴代码前点击左侧工具栏的“贴代码”按钮（
    ），然后在出现的“enter code here”处贴入你的代码。
  • 回复特定回复的时候，请点击该回复右侧的回复链接，系统将自动创建楼号和@通知。
  • @他人的时候注意，id后面请加一个空格。
  • 使用左侧工具栏“贴链接”按钮（
    ）创建的优酷、土豆、youtube视频链接，将自动生成播放区域，不需要使用其他包含方式。
  • 如果要创作长篇格式复杂的帖子，本站建议Mac用户使用Mou软件，离线写好，贴入即可。
